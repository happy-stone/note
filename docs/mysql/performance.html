性能

explain 计划

explain 命令可以验证 MySQL 的执行计划，查看查询优化器如何执行查询计划。

可以为正在执行的会话执行 explain 计划，需要制定 connection id（select connection_id() 查看当前 ID）。

索引

索引可以在执行查询操作时，快速找到数据，而不必扫描整个表。
